FORECAST LOGIC

透明预测逻辑

  1. 预测周销量 = W-4×10%＋W-3×20%＋W-2×30%＋最近一周×40%。
  2. 下周期预测销量 = 预测周销量×4；预计期末库存最低按0展示。
  3. 覆盖天数 = 当前库存÷预测周销量×7；28–56天暂作为演示合理区间。
  4. 达到补货点时，以6周目标库存计算建议补货量。

CSV SCHEMA

导入模板字段

  • 产品：product_code、product_name、sku_code、sku_name。
  • 库存：current_stock、unit_cost、lead_time_days、safety_stock。
  • 销量：sales_w4、sales_w3、sales_w2、sales_w1，其中W1为最近一周。
  • 字段名必须与模板一致；空行会忽略，数值字段不能为负数。
